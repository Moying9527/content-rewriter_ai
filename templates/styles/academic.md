# 课程作业/论文规则模板

## 适用场景

- 学术论文
- 课程作业
- 研究报告
- 毕业论文

## AI降味设置

**执行降味**：❌ **否**（保持学术规范）

**说明**：学术论文需要严格遵循学术规范，不执行AI降味处理，仅做格式规范化。保留学术术语和正式表达。

## 段落规则

| 维度 | 规则 |
|------|------|
| 段落长度 | 3-8行 |
| 段落开头 | 首行缩进2格 |
| 标题层级 | 一、→（一）→1.→（1） |
| 层级限制 | 不超过4级标题 |

## 标点规则

| 标点 | 规则 |
|------|------|
| 引号 | 双引号""，句号在引号外 |
| 省略号 | 一般不使用 |
| 破折号 | —— |
| 感叹号 | **禁止**使用（直接引用除外） |
| 问号 | 谨慎使用 |

## 数字规则

- 十以下用中文数字（一、二、三...）
- 十以上用阿拉伯数字（10、20、100）
- 统计数据、年份用阿拉伯数字

## 标题格式

| 层级 | 格式 | 示例 |
|------|------|------|
| 一级 | 一、 | 一、绪论 |
| 二级 | （一） | （一）研究背景 |
| 三级 | 1. | 1. 国内外现状 |
| 四级 | （1） | （1）理论研究 |

## 特殊处理

### 学术规范
- 引用格式正确（按学校/期刊要求）
- 参考文献格式统一
- 术语使用规范
- **保留学术术语和正式表达**

### 严谨性
- 论证逻辑严密
- 数据准确可靠
- 避免主观臆断

### 客观性
- 避免情绪化表达
- 用学术语言论述
- **禁止感叹号**

### 结构完整
- 摘要、关键词、正文、参考文献齐全
- 层次分明
- 过渡自然

## 论文特有元素

### 摘要
- 简明扼要
- 包含研究目的、方法、结果、结论
- 200-300字

### 关键词
- 3-5个
- 用分号分隔

### 参考文献
- 格式统一
- 按引用顺序或字母顺序排列

## 排版风格

- 学术规范
- 严谨客观
- 逻辑严密
- 格式统一
- **不降味，保持学术严谨性**

## 特别注意

- **感知动词保留**：学术论文中"认为""指出""发现"等是正常学术表达，不删除
- **连接词保留**："首先""其次""此外"等是论文常用结构词，保留
- **术语保留**：所有专业术语保持原样
- **引用保留**：直接引用的内容不做任何修改
