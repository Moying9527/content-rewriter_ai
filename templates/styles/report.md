# 正式报告/商务文档规则模板

## 适用场景

- 商务报告
- 工作汇报
- 正式文档
- 企业文书

## AI降味设置

**执行降味**：❌ **否**（保持严谨规范）

**说明**：报告类文本需要保持专业性和严谨性，不执行AI降味处理，仅做格式规范化。

## 段落规则

| 维度 | 规则 |
|------|------|
| 段落长度 | 5-8行 |
| 段落开头 | 首行缩进2格 |
| 标题层级 | 一、→（一）→1.→（1） |
| 层级限制 | 不超过4级标题 |

## 标点规则

| 标点 | 规则 |
|------|------|
| 引号 | 双引号""，句号在引号外 |
| 省略号 | ……（正式场合可用） |
| 破折号 | —— |
| 感叹号 | **禁止**使用（过于情绪化） |
| 问号 | 谨慎使用，避免反问 |

## 数字规则

- 统一使用阿拉伯数字
- 统计数据、百分比用阿拉伯数字
- 日期格式统一（如：2026年4月7日）

## 标题格式

| 层级 | 格式 | 示例 |
|------|------|------|
| 一级 | 一、 | 一、项目概述 |
| 二级 | （一） | （一）市场分析 |
| 三级 | 1. | 1. 目标客户 |
| 四级 | （1） | （1）年龄分布 |

## 特殊处理

### 严谨性
- 术语准确
- 数据真实
- 逻辑清晰
- **保留专业术语和正式表达**

### 客观性
- 避免主观情绪词
- 用事实和数据说话
- **禁止感叹号**

### 专业性
- 保留专业术语
- 统一术语使用
- **不过度口语化**

### 结构清晰
- 层次分明
- 过渡自然
- 结论明确

## 排版风格

- 严谨统一
- 专业规范
- 逻辑清晰
- 正式得体
- **不降味，保持原文专业性**
