# 标点快速修复硬规则（P0优先级）

> **执行顺序**：P0（必须执行）→ P1 → P2 → 降味处理
> 
> **执行原则**：优先级最高，先执行，无条件修复
> 
> **适用范围**：所有文体统一执行

## 英文标点 → 中文标点

| 错误 | 正确 | 示例 |
|------|------|------|
| 英文句号 `.` | 中文句号 `。` | `今天天气很好.` → `今天天气很好。` |
| 英文逗号 `,` | 中文逗号 `，` | `你好,世界` → `你好，世界` |
| 英文问号 `?` | 中文问号 `？` | `你在做什么?` → `你在做什么？` |
| 英文感叹号 `!` | 中文感叹号 `！` | `太棒了!` → `太棒了！` |
| 英文冒号 `:` | 中文冒号 `：` | `注意:以下` → `注意：以下` |
| 英文分号 `;` | 中文分号 `；` | `第一;第二` → `第一；第二` |
| 英文括号 `()` | 中文括号 `（）` | `(说明)` → `（说明）` |
| 英文引号 `""` | 中文引号 `""` | `"你好"` → `"你好"` |

## 省略号规范

| 错误 | 正确 | 示例 |
|------|------|------|
| 三个点 `...` | 省略号 `……` | `等等...` → `等等……` |
| 六个点 `......` | 省略号 `……` | 已是正确格式 |
| 连续句号 `。。。。` | 省略号 `……` | `他说。。。。` → `他说……` |
| 英文省略号 `...` | 省略号 `……` | `Well...` → `嗯……` |

## 破折号规范

| 错误 | 正确 | 示例 |
|------|------|------|
| 两个短横 `--` | 破折号 `——` | `北京--上海` → `北京——上海` |
| 单个短横 `-` | 破折号 `——`（视语境） | `解释-说明` → `解释——说明` |

## 引号规范

| 场景 | 规则 | 示例 |
|------|------|------|
| 对话 | 双引号 `""` | `"你好。"他说。` |
| 引用 | 双引号 `""` 或书名号《》 | `正如鲁迅所说："……"` |
| 特殊含义 | 双引号 `""` | `所谓的"专家"` |
| 内心独白（小说） | 单引号 `''` 或不加引号 | `'我该怎么办' / 我该怎么办` |

## 删除/替换符号

| 错误 | 处理 | 示例 | 例外情况 |
|------|------|------|----------|
| 波浪号 `~` | **删除** | `开心~` → `开心` | 小说刻意拖音保留（如：好~~~） |
| 连续感叹号 `!!!` | 保留1个 | `太棒了!!!` → `太棒了！` | 无 |
| 连续问号 `???` | 保留1个 | `什么???` → `什么？` | 无 |
| 混合标点 `？！` `！？` | 保留1个 | `真的？！` → `真的？` 或 `真的！` | 无 |
| 多余空格 | 删除 | `你好  世界` → `你好世界` | 英文前后空格按文体规则 |

## 空格规范

| 场景 | 小说/网文 | 公众号 | 报告 | 论文 |
|------|-----------|--------|------|------|
| 中英文之间 | 不加 | **加空格** | 不加 | 不加 |
| 数字与单位之间 | 加空格 | 加空格 | 加空格 | 加空格 |
| 标点符号后 | 不加 | 不加 | 不加 | 不加 |
| 段落开头 | 顶格 | 顶格 | 空两格 | 空两格 |

## 特殊标点限制

### 分隔线 `---`

> ⚠️ **小说分隔线使用规范（新增）**

**核心原则**：分隔线用于**大场景转换**，不是段落分隔符

**使用规则**：

| 文体 | 最大使用次数 | 使用场景 | 禁止场景 |
|------|-------------|----------|----------|
| **小说/网文** | **≤3处/章** | 地点大转换、时间大跳跃 | 段落过渡、情绪转折、小场景切换 |
| 公众号 | ≤1处/篇 | 章节分割 | 禁止 |
| 报告/论文 | ≤1处/篇 | 章节分割 | 禁止 |

**小说场景转换的正确方式**：

```
✅ 正确示例（小说）：
段落1内容。
段落2内容。

[空一行 = 一般过渡/情绪延续]

下一段内容。

*** [空一行] = 场景大幅转换（每章限1-2次）

新场景开始。

--- [空一行] = 地点/时间大跳转（每章≤3处）

新地点新时间的内容。
```

```
❌ 错误示例：
第一段内容。
---
第二段内容。  [滥用分隔线]
---
第三段内容。 [频繁使用影响阅读]
```

**章节内部结构规范**：

```
章节标题
          [空2行]

正文第一段

正文第二段

--- [大场景跳转，如：深圳→新疆]

正文内容

*** [同一地点内的场景转换，如：机场→大巴]

正文内容

--- [再次大场景跳转]

正文内容

**【本章完】**
```

**检查清单**：

```
执行完润色后，检查分隔线使用：
- [ ] 小说章节内分隔线 ≤3处
- [ ] 分隔线仅用于大场景/大时间跳转
- [ ] 小场景过渡用空行，不用分隔线
- [ ] 情绪/心理转折用空行，不用分隔线
- [ ] 章节信息不放正文内（单独文档或开头）
```

### 省略号 `……`
- 小说：可正常使用
- 公众号：≤2次/篇
- 报告/论文：一般不使用

### 感叹号 `！`
- 小说：克制使用（≤5次/章）
- 公众号：适度使用
- 报告/论文：**禁止使用**

### 书名号《》
- 书籍、文章、报刊、影视剧等使用书名号
- 课程、活动、比赛等不使用书名号（用引号）

## 标点与降味的协同

### 标点修复优先于降味
1. 先执行P0标点修复
2. 再执行降味处理
3. 降味过程中注意保持标点正确

### 特殊处理
- 降味拆分长句时，确保新句子标点正确
- 删除AI词汇后，检查标点衔接
- 对话口语化时，注意问号、感叹号的使用

## 修复检查清单

执行完P0规则后，检查以下项目：

- [ ] 无英文标点残留
- [ ] 省略号格式正确（……）
- [ ] 破折号格式正确（——）
- [ ] 引号使用规范（统一风格）
- [ ] 无多余波浪号（小说拖音除外）
- [ ] 无连续重复标点（!!!、???）
- [ ] 空格使用符合文体规则
- [ ] 分隔线使用符合限制（小说≤3处）
- [ ] 感叹号使用符合文体限制
- [ ] 小说章节信息不放在正文内
