# AI降味专项规则

> **执行条件**：小说/网文、公众号/自媒体默认执行；报告/论文不执行
> 
> **执行原则**：保留原文核心信息，不改变事实和逻辑，仅优化表达方式

## 降味目标

消除AI写作的典型痕迹：
1. 刻板句式、匀速平铺
2. 空洞套话、高频套路词
3. 过度完美的长句
4. 机械化的过渡连接
5. 情感缺失的陈述

注入人类写作特征：
1. 情感颗粒度、自然起伏
2. 口语节奏、日常习惯表达
3. 不规则表达、个性化语言
4. 长短句交替、呼吸感

## 降味规则体系

### 规则1：删除高频AI词汇

#### 1.1 套路过渡词

| AI高频词 | 替换建议 | 示例 |
|----------|----------|------|
| 值得注意的是 | 重点是、说白了、其实 | "值得注意的是，这个问题很关键" → "重点是，这个问题很关键" |
| 不难发现 | 一眼就能看出、明摆着 | "不难发现，数据有问题" → "明摆着，数据有问题" |
| 基于以上 | 说到底、归根结底 | "基于以上分析" → "说到底" |
| 总的来说 | 整体来看、反正 | "总的来说，效果不错" → "整体来看，效果不错" |
| 首先…其次…再次… | 一个是…再一个…最后 | 直接分段或简化连接 |
| 总而言之 | 说白了、一句话 | "总而言之，我们成功了" → "一句话，我们成功了" |
| 毫无疑问 | 明摆着、不用说 | "毫无疑问，这是对的" → "明摆着，这是对的" |
| 众所周知 | 大家都知道、提起这个 | "众所周知，地球是圆的" → "大家都知道，地球是圆的" |
| 从某种程度上说 | 说真的、讲真 | "从某种程度上说，他是对的" → "说真的，他是对的" |
| 具有重要意义 | 很关键、这点很重要 | "这具有重要意义" → "这很关键" |
| 随着社会的不断发展 | 现在、这些年 | "随着社会的不断发展" → "这些年" |
| 引发了广泛的关注 | 很多人都注意到了 | "引发了广泛的关注" → "很多人都注意到了" |
| 本文主要论述 | 今天聊聊、这篇文章说 | "本文主要论述" → "这篇文章说" |
| 可以说 | 说起来 | "可以说，这是最好的" → "说起来，这是最好的" |
| 可见 | 看得出来 | "可见，他很努力" → "看得出来，他很努力" |
| 实际上 | 说起来、其实 | "实际上，情况很复杂" → "其实，情况很复杂" |
| 必须指出 | 得说、得提一下 | "必须指出，这里有错误" → "得说，这里有错误" |
| 毫无疑问地 | 肯定的、没问题 | "毫无疑问地，他会来" → "肯定的，他会来" |
| 值得强调的是 | 关键在这儿、得记住 | "值得强调的是" → "关键在这儿" |
| 除此之外 | 再说、还有 | "除此之外，还有一件事" → "还有一件事" |
| 换句话说 | 白话说就是、说白了 | "换句话说，就是失败了" → "说白了，就是失败了" |
| 综上所述 | 总之、反正 | "综上所述，我们得出结论" → "总之，我们得出结论" |
| 因此可以看出 | 一看就知道 | "因此可以看出" → "一看就知道" |

#### 1.2 感知动词（小说场景重点删除）

| 感知动词 | 处理方式 | 示例 |
|----------|----------|------|
| 感到 | 直接写感受 | "他感到一阵寒意" → "一阵寒意袭来" |
| 意识到 | 直接呈现认知 | "她意识到危险" → "危险近在咫尺" |
| 注意到 | 直接描述 | "他注意到异常" → "异常显而易见" |
| 发现 | 直接陈述 | "她发现秘密" → "秘密浮出水面" |
| 觉得 | 直接表达 | "他觉得不对" → "不对" |
| 认为 | 视文体决定（论文保留） | 小说删除，论文保留 |

#### 1.3 机械连接词

| 连接词 | 处理方式 | 示例 |
|--------|----------|------|

| 紧接着 | 删除或替换 | "紧接着发生了" → "发生了" |
| 随后 | 删除或替换 | "随后跟了上去" → "跟了上去" |

#### 1.4 冗余副词

| 副词 | 处理方式 | 示例 |
|------|----------|------|
| 微微 | 删除 | "微微颤抖" → "颤抖" |
| 稍稍 | 删除 | "稍稍抬头" → "抬头" |
| 略显 | 删除 | "略显疲惫" → "疲惫" |
| 缓缓 | 删除 | "缓缓坐下" → "坐下" |
| 渐渐 | 删除 | "渐渐明白" → "明白" |
| 逐渐 | 删除 | "逐渐适应" → "适应" |
| 开始 | 删除 | "开始说话" → "说话" |
| 开始变得 | 删除 | "开始变得紧张" → "紧张" |
| 某种 | 具体化 | "某种感觉" → "奇怪的感觉" |
| 一种 | 具体化 | "一种情绪" → "压抑的情绪" |
| 一股 | 具体化 | "一股力量" → "强大的力量" |

#### 1.5 心理引导词

| 引导词 | 处理方式 | 示例 |
|--------|----------|------|
| 不禁 | 删除 | "他不禁想" → 直接写想法 |

| 本能地 | 视情况 | "她本能地躲开" → "她躲开" |

### 规则2：句式重构

#### 2.1 长句拆分

**拆分标准**：超过35字的句子建议拆分

**拆分方法**：
- 按意思完整性拆分
- 拆成短句群（2-4句）
- 保留核心信息

**示例**：
- ❌ "他慢慢地推开门，然后小心翼翼地走了进去，环顾四周后发现房间里空无一人。"（42字）
- ✅ "他推开门。走进去。环顾四周。房间里空无一人。"

#### 2.2 完美长句改造

**AI特征**：主谓宾定状补俱全的完美长句

**改造方法**：
- 删除冗余修饰
- 拆分多个短句
- 保留核心动作

**示例**：
- ❌ "在昏暗的灯光下，他小心翼翼地推开了那扇沉重的木门，然后慢慢地走了进去。"
- ✅ "灯光昏暗。他推开门，走进去。"

#### 2.3 匀速平铺打破

**AI特征**：段落长度高度均匀，节奏单调

**打破方法**：
- 制造长短句交替
- 插入短促的独词句
- 使用口语断句

**示例**：
- ❌ "他走进房间。他打开灯。他坐在椅子上。他开始思考。"
- ✅ "他走进房间，打开灯。坐下。思考。"

### 规则3：对话优化

#### 3.1 对话引导词简化

| 原表达 | 优化 | 示例 |
|--------|------|------|
| 他说道 | 他说 / 直接省略 | "他说道：'你好'" → "他说：'你好'" / "'你好。'" |
| 她问道 | 她问 / 直接省略 | "她问道：'为什么'" → "她问：'为什么'" / "'为什么？'" |
| 他回答道 | 他说 / 直接省略 | 简化或省略 |
| 他大声说道 | 他说，+动作 | "他大声说道" → "他提高了音量" |

#### 3.2 对话口语化

**加入口语元素**：
- "嗯""啊""那个""是吧"
- 不完整的句子
- 重复、停顿

**示例**：
- ❌ "他说：'我认为这个方案是可行的，我们可以尝试一下。'"
- ✅ "他说：'我觉得……这个方案行，可以试试。'"

#### 3.3 对话分段

**规则**：
- 每个说话者单独成段
- 动作可与对话同段
- 不同说话者之间空行

### 规则4：心理活动优化

#### 4.1 直接内心独白

**删除引导词**：
- ❌ "他想：完了完了"
- ✅ "完了完了"

- ❌ "她心想：该怎么办"
- ✅ "该怎么办"

#### 4.2 内心独白不加引号

**规则**：内心独白直接呈现，不加引号（小说场景）

**示例**：
- ❌ "他心想：'我不能放弃。'"
- ✅ "不能放弃。"

### 规则5：情绪表达动作化

#### 5.1 用动作代替情绪词

| 原表达 | 优化 | 示例 |
|--------|------|------|
| 他很愤怒 | 动作描写 | "他握紧拳头，额角青筋暴起" |
| 她很害怕 | 动作描写 | "她缩了缩肩膀，不敢抬头" |
| 他很紧张 | 动作描写 | "他不停地搓着手，眼神飘忽" |
| 她很伤心 | 动作描写 | "她低下头，眼泪滴在手背上" |
| 他很惊讶 | 动作描写 | "他瞪大了眼睛，嘴巴微张" |

#### 5.2 感官细节注入

**增加感官描写**：
- 视觉：颜色、光影、细节
- 听觉：声音、音量、音质
- 嗅觉：气味、香臭
- 触觉：温度、质地、触感
- 味觉：味道、口感

**示例**：
- ❌ "房间里很压抑。"
- ✅ "房间里弥漫着潮湿的霉味，空气沉重得让人喘不过气。"

#### 5.3 感官描写碎片化（★经验总结新增★）

> AI的感官描写是"罗列式"的，润色后应该变成"碎片式"

**罗列式 → 碎片式**：
- ❌ "空气里有泥土的味道，有青草的味道，还有炊烟的味道。"
- ✅ "空气里头有股子泥土味儿，有股子青草味儿，还混着股子炊烟味儿。"

**关键改动**：
- "味道" → "味儿"（加儿化音）
- "有……的" → "有股子……"（加量词）
- 混合用"混着"替代"还有"，更自然

### 规则6：过渡自然化

#### 6.1 替换机械过渡

| 机械过渡 | 自然过渡 | 示例 |
|----------|----------|------|
| 首先…其次…再次… | 直接分段或简化 | 删除过渡词，直接陈述 |
| 一方面…另一方面… | 不过、可是、其实 | "一方面他很努力，另一方面他很笨" → "他很努力，不过也很笨" |
| 虽然…但是… | 松散表达 | "虽然很累，但是坚持" → "累是累，但还在坚持" |
| 因为…所以… | 直接因果 | "因为下雨，所以迟到" → "下雨了，迟到了" |
| 既然…就… | 口语化 | "既然来了，就坐下" → "来都来了，坐吧" |

#### 6.2 场景转换自然化

**优先使用**：
- 动作转换："他站起身，走向门口"
- 时间转换："三天后"
- 空间转换："与此同时，在另一个城市"

**避免使用**：
- ❌ "让我们把视线转向…"
- ❌ "接下来，我们来看看…"

#### 6.3 景物描写去掉主语（★经验总结新增★）

> AI描写场景时喜欢加主语："她看见……她注意到……她发现……"
> 润色后直接删掉主语，让景物自己"跳出来"

**示例**：
- ❌ "她看见天空从灰蒙蒙变成淡淡的蓝色，看着第一缕阳光……"
- ✅ "瞅着天色从灰蒙蒙变成淡淡的蓝，瞅着第一缕日头从东边的山头后头探出来……"

**规律**：去掉"她看见/她注意到"，直接描写，视角自然跟随主角。

### 规则7：个性化表达

#### 7.1 允许不规则表达

**允许**：
- 不完整的句子
- 突然的转折
- 语序的倒装
- 重复和冗余（适度）

**示例**：
- ✅ "来了。终于来了。"
- ✅ "他，走了。"
- ✅ "不知道。真的不知道。"

#### 7.2 方言和口头禅（适度）

**小说场景**：
- 可适当使用方言词
- 加入人物口头禅
- 保持人物语言特征

#### 7.3 模糊词适度使用

**允许**：
- "似乎""好像""有点"（关键处保留）
- 不过度堆砌即可

## 降味强度分级

| 级别 | 适用场景 | 执行内容 |
|------|----------|----------|
| **基础降味** | <200字文本 | 删除高频AI词、感知动词、机械连接词 |
| **标准降味** | 200-1000字 | 基础降味 + 长句拆分、对话优化 |
| **完整降味** | >1000字 | 标准降味 + 心理活动优化、情绪动作化、过渡自然化、个性化表达 |

## 文体差异化降味策略

| 文体 | 降味执行 | 特殊处理 |
|------|----------|----------|
| 小说/网文 | ✅ 完整降味 | 重点删除感知动词、心理引导词；保留文学性 |
| 公众号 | ✅ 完整降味 | 重点删除套路词；保留情绪饱满；金句突出 |
| 报告 | ❌ 不执行 | 仅格式规范；保留专业术语 |
| 论文 | ❌ 不执行 | 仅格式规范；保留学术表达；感知动词保留 |

## 字数约束规则（硬性要求）

### 核心原则

**润色后正文字数不得低于原文的 95%**

### 字数检测流程

1. **处理前统计**：记录原文总字数（不含标点、空格）
2. **处理后统计**：记录润色后正文总字数
3. **计算比例**：润色后字数 ÷ 原文字数 × 100%
4. **达标判断**：
   - ≥ 95%：通过，正常输出
   - < 95%：触发字数补偿机制

### 字数补偿机制

当润色后字数低于 95% 时，按以下优先级补充内容：

1. **扩展感官细节**：增加视觉、听觉、嗅觉、触觉、味觉描写
2. **丰富动作描写**：细化人物动作，增加动作连贯性
3. **补充环境描写**：增加场景氛围、天气、光线等细节
4. **深化心理活动**：扩展内心独白，增加情绪层次
5. **添加对话内容**：适当增加对话，丰富人物互动

### 字数控制技巧

**删除AI词时的补偿策略**：
- 删除"感到" → 补充具体感受描写
- 删除"然后" → 补充动作细节或场景过渡
- 删除"微微" → 补充具体程度描写
- 删除长句拆分后的冗余 → 每拆分一处，补充一处细节

**示例**：
- 原文："他感到一阵寒意，然后推开了门。"（14字）
- 初版润色："他推开门。"（5字，35.7%，不达标）
- 补偿后："一阵寒意从脊背爬上来。他推开门，门轴发出吱呀一声。"（26字，185%，达标）

## 降味检查清单

执行完降味规则后，检查以下项目：

- [ ] 无高频AI词汇残留
- [ ] 感知动词已删除（论文除外）
- [ ] 机械连接词已删除或替换
- [ ] 冗余副词已删除
- [ ] 长句已适当拆分（超过35字）
- [ ] 对话已口语化
- [ ] 心理活动已直接呈现
- [ ] 情绪已动作化
- [ ] 过渡已自然化
- [ ] 无新的"AI味"引入
- [ ] 原文核心信息保留完整
- [ ] **字数达标：润色后字数 ≥ 原文字数 × 95%**
