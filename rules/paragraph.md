# 段落调整规则（P1/P2优先级）

> **执行顺序**：P0（标点修复）→ **P1（段落长度）→ P2（标题层级）** → 降味处理
> 
> **执行原则**：按文体模板执行，不改变原文核心内容
> 
> **与降味的协同**：段落调整可能触发降味（如对话分段后的口语化）

## P1：段落长度调整

### 段落长度标准

| 文体 | 标准长度 | 极短段落 | 超长段落 |
|------|----------|----------|----------|
| 小说/网文 | 2-6行 | 1-2行（情绪/转折） | 仅用于紧张/压抑场景 |
| 公众号 | 2-4行 | 1行（金句/强调） | 避免超过5行 |
| 报告 | 5-8行 | 3-4行（过渡） | 可适当延长 |
| 论文 | 3-8行 | 2-3行（过渡） | 可适当延长 |

### 段落拆分规则

**需要拆分的情况**：
1. 段落超过文体上限（如公众号超过5行）
2. 一个段落包含多个独立意思
3. 对话与叙述混排
4. 金句被埋没在长段落中
5. 包含多个动作或场景

**拆分方法**：
- 按意思完整性拆分
- 对话单独成段
- 动作与对话可同段
- 不同说话者用空行分隔
- 金句单独成段

**拆分示例**（小说场景）：
- ❌ "他推开门走了进去，环顾四周发现房间里空无一人，心里有些不安，但还是决定继续往前走。"
- ✅ "他推开门，走进去。环顾四周。房间里空无一人。他心里有些不安，但还是决定继续往前走。"

### 段落合并规则

**需要合并的情况**：
1. 连续多个1行段落（非对话/金句）
2. 意思紧密相关的短句
3. 过度碎片化影响阅读

**合并方法**：
- 保持逻辑连贯
- 保留关键过渡词
- 注意节奏变化

**合并示例**：
- ❌ "他来了。他看到了。他征服了。"
- ✅ "他来了，看到了，征服了。"

### 空行使用规范

| 场景 | 空行数 |
|------|--------|
| 段落之间 | 1空行 |
| 章节之间 | 2空行 |
| 场景转换 | 2空行或 `***` |
| 分隔线前后 | 1空行 |
| 不同说话者之间 | 1空行 |

## P2：标题层级调整

### 报告/论文标题层级

```
一级标题：一、标题内容
二级标题：（一）标题内容
三级标题：1. 标题内容
四级标题：（1）标题内容
```

**规则**：
- 层级不超过4级
- 同级标题格式统一
- 标题后不加标点（除顿号、括号）
- 标题与正文间1空行

### 公众号标题层级

```
一级标题：## 标题内容（或加粗）
二级标题：**小标题**（加粗）
三级标题：- 要点
```

**规则**：
- 可用Markdown格式
- 可加emoji增强视觉
- 金句可单独成段并加粗
- 小标题简短有力

### 小说章节标题

```
第一章 标题

或

第1章 标题
```

**规则**：
- 统一使用中文或阿拉伯数字
- 标题后2空行
- 可用 `***` 分隔场景

## 特殊排版规则

### 对话排版

**小说场景**：
```
"你来了。"他说。

"嗯。"她点点头。
```

**规则**：
- 对话单独成段
- 说话者动作可与对话同段
- 多人对话用空行分隔
- 对话口语化（配合降味）

### 列表排版

**有序列表**：
```
1. 第一点
2. 第二点
3. 第三点
```

**无序列表**：
```
- 第一点
- 第二点
- 第三点
```

**规则**：
- 列表项之间1空行
- 列表前后各1空行
- 列表项不宜过长

### 引用排版

```
> 引用内容
> 多行引用
```

**规则**：
- 引用前后各1空行
- 引用内部保持段落结构
- 长引用可分段

## 文体特殊处理

### 小说特殊规则

- 保留作者个性化风格
- 允许适度留白
- 场景转换可用 `***` 或空行
- 心理描写不加引号
- **对话分段后配合降味口语化**

### 公众号特殊规则

- 金句单独成段
- 重要内容可加粗
- 保留emoji
- 英文前后加空格
- **短段落配合降味删除套路词**

### 报告特殊规则

- 首行缩进2格
- 标题层级严格规范
- 禁止感叹号
- 数据用阿拉伯数字
- **不降味，保持专业**

### 论文特殊规则

- 首行缩进2格
- 摘要、关键词、参考文献格式规范
- 引用格式统一
- 术语准确
- **不降味，保持学术严谨**

## 段落调整与降味的协同

### 拆分触发降味

当段落被拆分时，可能触发降味处理：

1. **对话分段** → 自动口语化
   - ❌ "他说道：'我认为这个方案可行。'"
   - ✅ 分段后 → "他说：'我觉得这方案行。'"

2. **动作拆分** → 删除连接词
   - ❌ "然后他推开门，接着走了进去。"
   - ✅ 拆分后 → "推开门。走进去。"

3. **心理活动拆分** → 删除引导词
   - ❌ "他想：我该怎么办。"
   - ✅ 拆分后 → "该怎么办。"

### 合并保留降味

段落合并时，保留已执行的降味效果：
- 合并后检查是否引入新的AI词汇
- 确保合并后的句子自然流畅

## 调整检查清单

执行完P1/P2规则后，检查以下项目：

- [ ] 段落长度符合文体标准
- [ ] 无过长或过短段落（特殊情况除外）
- [ ] 标题层级正确
- [ ] 空行使用规范
- [ ] 对话排版正确
- [ ] 列表排版正确
- [ ] 引用排版正确
- [ ] 段落调整后未引入AI词汇
- [ ] 整体排版美观易读
